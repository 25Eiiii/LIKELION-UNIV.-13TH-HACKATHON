# 모락모락
**지역 문화가 피어나는 공간**  
> 지역 주민이 흩어진 문화행사 정보를 한 곳에서 쉽게 찾고, AI 맞춤 추천을 통해 나에게 딱 맞는 문화 활동을 발견·참여할 수 있도록 돕는 서비스입니다.

## 프로젝트 소개

모락모락은

- 지역 문화 행사 정보 통합

- AI 기반 맞춤형 추천

- 참여 리워드(포인트 → 쿠폰)

- 제휴 상점 혜택 제공

까지 하나로 묶은 지역문화 활성화 플랫폼입니다.

## 기술 스택

### 프론트엔드
- React

### 백엔드
- Django
- PostgreSQL
- Railway / Redis

### AI & Recommendation
- 콘텐츠 기반 필터링 (TF-IDF + 코사인 유사도)
